# 📊 Vrinda Store Sales Dashboard (Microsoft Excel)

An interactive **Sales Analysis Dashboard** built in **Microsoft Excel** to analyze Vrinda Store sales performance. This project transforms raw sales data into meaningful business insights using Pivot Tables, Pivot Charts, Slicers, and Dashboard Design.

---

## 📌 Project Overview

The objective of this project is to analyze sales data and create an interactive dashboard that helps understand:

- Monthly Sales Performance
- Order Trends
- Customer Demographics
- Top Performing States
- Sales Channels
- Order Status
- Product Category Analysis

The dashboard enables users to filter data dynamically and make data-driven decisions.

## 📌 Key Business Insights

- Women contribute the highest percentage of overall sales.
- Most orders are successfully delivered.
- Maharashtra generates the highest sales.
- Amazon is the top-performing sales channel.
- Adult customers contribute the highest number of orders.
- Sales peak during the first quarter of the year.
